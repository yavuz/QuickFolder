# Security Policy

QuickFolder is a local macOS utility. It stores folder metadata locally and does not intentionally send folder paths or user data to a network service.

## Reporting Security Issues

Please do not open a public issue for sensitive security reports.

Use GitHub private vulnerability reporting if it is enabled for the repository, or contact the repository owner directly.

## Data Handling

- Pinned and recent folder metadata is stored locally.
- Security-scoped bookmark data may be stored to preserve folder access.
- macOS recent-folder import is best-effort and read-only.
